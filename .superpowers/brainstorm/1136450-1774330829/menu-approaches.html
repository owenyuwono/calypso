Menu Consolidation — 3 Approaches

How should we restructure the 7 keyboard panels into fewer, more complete ones?

A

Tabbed Character Sheet (Recommended)

3 panels total. One unified Character panel (Tab key) with internal tabs: Stats | Equipment | Skills. Quest Log (J) and Settings (Esc) stay separate. World Map stays on minimap compass.

╔══════════════════════════════════════╗
📊 Stats ⚔️ Equip ✨ Skills
OFFENSIVE
ATK: 42   MATK: 18
Accuracy: 90   Crit: 7%
DEFENSIVE
HP: 125/125   DEF: 28
MDEF: 12   Evasion: 9
── Proficiency Levels ──
Sword ★★★ | STR ★★ | Mining ★★★★
╚══════════════════════════════════════╝

Pros

  • Only 3 keybinds: Tab, J, Esc
  • All character info in one place
  • Tabs feel natural and modern
  • Easy to add future tabs

Cons

  • Larger panel = more screen coverage
  • Slightly more clicks to reach specific info
B

JRPG Full-Screen Menu

1 panel. Single Esc/Tab opens a full-screen menu with sidebar: Status, Items, Skills, Quests, Map, Settings. Everything in one place.

MENU
▸ Status
  Items
  Skills
  Quests
  Map
  Settings
CHARACTER STATUS
HP: 125/125   ATK: 42   DEF: 28
MATK: 18   MDEF: 12
Proficiencies, equipment summary...

Pros

  • 1 keybind for everything
  • Classic RPG feel (Final Fantasy, Persona)
  • Maximum consolidation

Cons

  • Full-screen = can't see the game world
  • Doesn't fit the MMO/RO aesthetic
  • More navigation depth to reach anything
C

Smart Merge (4 panels)

4 panels. Character = Stats + Equipment + Inventory. Skills = Proficiencies + Active Skills + Hotbar. Quest Log stays. Settings stays. No tabs — each panel is a single view.

CHARACTER (C)
Stats + Equipment + Items
Left: equip slots + stats
Right: item grid
SKILLS (S)
Proficiencies + Skills + Hotbar
Prof list → drill-down
Hotbar assignment
QUESTS (J)
SETTINGS (Esc)

Pros

  • 4 keybinds: C, S, J, Esc
  • Each panel is self-contained, no tabs
  • Closest to current structure (less rework)

Cons

  • Character panel gets very dense
  • Still 4 keys to remember
  • Stats + inventory in one panel may feel crowded
